# 🏋️ Diário de Treino - Django

Sistema web desenvolvido com Django para auxiliar usuários no cadastro e organização de treinos semanais.

---

# 📌 Sobre o Projeto

O objetivo deste projeto é permitir que usuários possam:

- cadastrar treinos
- organizar rotina semanal
- visualizar exercícios
- gerenciar informações de treino

Este projeto foi desenvolvido com foco em aprendizado prático utilizando Django + SQLite3.

---

- Python
- Django
- SQLite3
- HTML
- CSS

# ⚙️ Como Executar o Projeto

## 1. Clone o repositório

```bash
git clone https://github.com/SEU_USUARIO/diario-treino-django.git
```

---

## 2. Acesse a pasta do projeto

```bash
cd diario-treino-django
```

---

## 3. Crie o ambiente virtual

### Windows

```bash
python -m venv venv
```

### Linux/Mac

```bash
python3 -m venv venv
```

---

## 4. Ative o ambiente virtual

### Windows

```bash
venv\Scripts\activate
```

### Linux/Mac

```bash
source venv/bin/activate
```

---

## 5. Instale as dependências

```bash
pip install django
```

---

## 6. Execute as migrations

```bash
python manage.py migrate
```

---

## 7. Inicie o servidor

```bash
python manage.py runserver
```

---

## 8. Acesse no navegador

```bash
http://127.0.0.1:8000/
```
